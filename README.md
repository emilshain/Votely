# Votely

Votely is a web application that allows users to create and manage polls. It provides a user-friendly interface for creating polls, adding options, and managing poll settings. The application also includes features for voting, viewing poll results, and exporting poll data.

## Features

- Create polls with multiple options
- Manage poll settings and options
- View poll results in real-time
- Export poll data as CSV
- Share polls with others
- Vote on polls created by others

## Technologies Used

- React
- Node.js
- Express
- SQLite
- Tailwind CSS
- Chart.js
